A equipa do Subsurface providencia uma grande variedade de ficheiros de instalação aos seus utilizadores. Alguns estão disponíveis como downloads directos a partir deste sítio e outros estão disponíveis através dos canais habituais das várias distribuições Linux.

A versão mais recente do Subsurface é a {{ site.latest }} (por vezes possa demorar alguns dias até que os vários ficheiros binários sejam actualizados e estejam disponíveis).

Windows

Os ficheiros de instalação do Subsurface suportam as versões de 32 e 64bits do Windows 7, 8, and 10. O suporte a computadores de mergulho que usem o protocolo Bluetooth Low Energy requer o Windows 10. O suporte MTP (presentemente apenas necessário para o Garmin Descent Mk2/Mk2i) requer a versão 64bit do Windows.

macOS

O Subsurface corre em Macs com processador Intel com o macOS 10.12 ou mais recente. Por razões técnicas, estão disponíveis três DMGs diferentes.

Linux

O elevado número de distribuições Linux e os diferentes formatos de pacotes tornam muito mais difícil a distribuição de binários para este sistema. No entanto, os desenvolvedores do Subsurface tentam disponibilizar ficheiros de instalação para as distribuições (e formatos de pacotes) mais populares:

Ubuntu e distribuições compatíveis

Para muitos utilizadores, o formato Snap pode ser a solução mais simples --- um pacote nativo pode ser instalado adicionando oppa:subsurface/subsurface PPA ao sistema.

Snap
Instalar via Snap

Fedora e openSUSE

Muitas versões correntes e ainda mantidas para várias arquitecturas estão disponíveis via OBS.

openSUSE/Fedora
Instalar via OBS

AppImage genérica

Se a sua distribuição não for suportada e se estiver a correr numa plataforma x86-64, uma AppImage é normalmente uma opção viável. Deverá descarregar este ficheiro, dar-lhe permissões de execução --- chmod +x Subsurface-{{ site.latest }}-x86_64.AppImage --- e depois simplesmente correr o ficheiro.

O código fonte dos componentes GPL e LGPL disponibilizados com os ficheiros binários para Mac e Windows podem ser encontrados em Homebrew e MXE. O código fonte do Subsurface vem acompanhado de todos os scripts necessários para compilar quer o Subsurface, quer as suas dependências para Windows, Mac e Linux.
