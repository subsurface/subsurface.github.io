El equipo de Subsurface pone a disposición de nuestros usuarios varios instaladores. Algunos de ellos están disponibles como descargas directas desde esta página, otros se pueden descargar a través de los mecanismos de distribución de Linux.

La última versión de Subsurface es {{ site.latest }} - en ocasiones lleva varios días actualizar todos los binarios.

Windows

Los instaladores de Subsurface soportan Windows 7, 8, y 10, en 32bit y 64bit. El soporte de Bluetooth Low Energy requiere Windows 10. El soporte de MTP (actualmente sólo necesario para el Garmin Descent Mk2/Mk2i) requiere 64bit.

macOS

Subsurface se ejecuta en Macs basados en Intel con macOS 10.12 o más reciente. Por motivos técnicos están disponibles tres DMGs distintos.

Linux

Linux complica mucho más ofrecer binarios dado el gran número de distribuciones disponibles con diferentes mecanismos de empaquetado. Los desarrolladores de Subsurface están intentando hacer paquetes para las distribuciones más populares (y en los formatos más populares):

Ubuntu y distros compatibles

Para muchos usuarios el Snap puede ser la solución más simple, se puede instalar un paquete nativo añadiendo la ppa:subsurface/subsurface PPA a tu sistema.

Snap
Install via Snap

Fedora and openSUSE

Aún se encuentran disponibles y mantenidas muchas versiones para diferentes arquitecturas vía OBS.

openSUSE/Fedora
Install via OBS

AppImage genérica

Si tu distribución no es una de las indicadas anteriormente y estás usando una plataforma x86-64, habitualmente una AppImage es una solución viable. Deberías poder descargar este archivo, hacerlo ejecutable chmod +x Subsurface-{{ site.latest }}-x86_64.AppImage y luego, simplemente, ejecutarlo.

AppImage
Download AppImage

Las fuentes de los componentes GPL y LGPL incluidos en las versiones Mac y Windows se pueden encontrar en Homebrew y MXE. El código fuente de Subsurface viene con todos los scripts necesarios para compilar Subsurface y sus dependencias para Windows, Mac y Linux.
