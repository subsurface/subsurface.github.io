L'équipe de Subsurface rend disponible une série d'installeurs pour les utilisateurs. Certains d'entre eux sont disponibles en tant que téléchargements directs, d'autres le sont via les mécanismes habituels de distribution des différentes distributions Linux.

La dernière version de Subsurface est {{ site.latest }} - généralement, il faut compter quelques jours pour que les différentes applications soient mises à jour.

Windows

Les installeurs de SubSurface supportent Windows 7, 8, et 10, tant en 32bit qu'en 64bit. Le support de Bluetooth Low Energy nécessite Windows 10. Le support de MTP (actuellement requis uniquement pour le Garmin Descent Mk2/Mk2i) nécessite une architecture 64bit.

macOS

Subsurface tourne sur des Macs à base Intel avec macOS 10.13 ou plus récent. Pour des motifs techniques, il y a deux images-disque DMG différentes disponibles.

Linux

Linux rend bien plus difficile d'offrir des applications vu le nombre important de distributions différentes, avec des mécanismes de distributions différents pour la réalisation d'applications utilisable. Les développeurs de Subsurface tentent de rendre disponibles des applications pour les distributions les plus populaires (et les formats les plus populaires):

Ubuntu and compatible distros

Pour de nombreux utilisateurs, Snap est la solution la plus simple, un paquet natif peut être installé en ajoutant le ppa:subsurface/subsurface PPA à votre système.

Snap
Install via Snap

Fedora and openSUSE

De nombreuses versions courantes et toujours maintenues pour plusieurs architectures sont disponibles via OBS.

openSUSE/Fedora
Install via OBS

Debian

Des paquets pour les versions récentes de Debian (généralement pour stable, testing, et unstable) pour les architectures amd64 et i386 sont disponibles via un dépôt tierce partie.

Debian packages
Usage instructions

generic AppImage

Si votre distribution n'est pas supportée ci-dessus et que vous utilisez une plateforme x86-64, une AppImage est généralement une solution valable. Vous devriez pouvoir télécharger ce fichier, le rendre exécutable chmod +x Subsurface-{{ site.latest }}-x86_64.AppImage et l'exécuter simplement.
